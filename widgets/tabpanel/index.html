多页标签面板（选项卡）

注意：
一个 Widget 本身仍是一个元素。
当一个元素成为 Widget 时，将获得新的属性、方法，具备新的行为，并能触发新的事件。
这些新增的特性并不妨碍将一个 Widget 视为一个普通元素来对其进行操作（如修改某部分的内容、结构、表现或行为）。

基本应用

  • 适用于绝大多数场景。
  • 不需要写脚本。

鼠标点击切换选项卡

按照结构约定来设计 HTML 即可。

鼠标悬停（200 毫秒）切换选项卡

通过可配置项 data-hover-delay 来指定鼠标悬停切换的时间，本例为 200 毫秒。

高级应用

  • 通过编写脚本来满足更多需求。

激活指定的标签

使用 activate 方法来激活指定的标签。

监听事件

可以为 activate 事件添加监听器。
