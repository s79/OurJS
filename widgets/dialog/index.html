模态对话框

注意：
一个 Widget 本身仍是一个元素。
当一个元素成为 Widget 时，将获得新的属性、方法，具备新的行为，并能触发新的事件。
这些新增的特性并不妨碍将一个 Widget 视为一个普通元素来对其进行操作（如修改某部分的内容、结构、表现或行为）。

对话框弹出时禁止对整个视口或指定元素的操作

  • 当对话框弹出时将自动生成一个遮盖层，遮盖层遮盖的范围为对话框父元素的渲染范围。被遮盖的部分将无法使用键盘或鼠标进行操作。
  • “模态”对话框的遮盖层是一定会出现的，不能将其屏蔽。
  • 当前打开的对话框会在遮盖层上方显示。

禁止对整个视口的操作

如果对话框元素的父元素是 BODY，遮盖层将遮盖整个视口。这种情况下当对话框弹出时，将只能对此对话框进行操作。

禁止对指定元素的操作

如果对话框元素的父元素不是 BODY，遮盖层将只遮盖其父元素。这种情况下当对话框弹出时，除此对话框外，还可以对页面中未被遮盖的部分进行操作。

跟随视口滚动或固定显示

  • 对话框元素的父元素是 BODY 时，可以通过设置对话框的 position 为 fixed 来使其固定显示。
  • 如果对话框元素的父元素不是 BODY，则无法固定其显示位置，只能跟随视口滚动。

跟随视口一起滚动

对话框默认情况下会跟随视口一起滚动。

视口滚动时位置保持不变

将对话框元素的 position 样式设置为 fixed 即可固定其显示位置。只有对话框的父元素是 BODY 时这样设置才有效。

指定对话框的显示位置

  • 默认情况下对话框的“定位参考元素”为其父元素，但可以通过 data-pinned-target 属性来指定其他的“定位参考元素”。
  • 默认情况下对话框的中心点与其“定位参考元素”的中心点重合，如果其“定位参考元素”为 BODY，则对话框的中心点与视口的中心点重合。

设置“定位参考元素”

通过配置对话框的 data-pinned-target 属性来指定其“定位参考元素”，“定位参考元素”只能是对话框的父元素或其父元素的后代元素。

设置偏移量

通过配置对话框的 data-offset-x 属性和 data-offset-y 属性来指定对话框的左边和顶边与其“定位参考元素”的左边和顶边的差值，指定的差值的单位是像素。

使用不同的动画效果

  • 可以通过配置对话框的 data-animation 属性来指定打开和关闭对话框时使用的动画效果。可选项有 'none'，'fade' 和 'slide'。

启用滑动效果 (slide)

多个对话框层叠显示

  • 当多个对话框有相同的父元素时，则视这些对话框为一组，一组对话框可以重叠显示。

两个对话框层叠

当一组对话框重叠显示时，遮盖层只有一个，且只有最后打开的对话框才可以操作。
